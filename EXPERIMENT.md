# Experimental PonsWarp Next UI

This branch keeps the Next.js `apps/web` frontend as an **experimental PonsWarp UI**.

- Production deployable UI: `PonsWarp/` on mainline (`master` / `feature/unified-workspace`)
- This UI: `apps/web/` on `experiment/ponswarp-next-ui` only
- Do **not** deploy this branch to `warp.ponslink.com`

## Run

```bash
cd apps/web
pnpm install
pnpm dev
```

Opens `http://localhost:3000`.
